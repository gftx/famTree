# famTree

##How to start:

1) you need to create .env file into server directory and add to it your database credentials
2) write in the terminal </br>
    ```cd client```</br>
   ```yarn install```</br>
   ```yarn start```</br>
   ```cd ../server```</br>
   ```yarn install```</br>
   ```yarn start```</br>
3) done, famtree is running on localhost:3000
